HTML Introduction

  • Estandar para hacer una pagina web (Standart for Hyper Text Markup Language)
  • Describe la estructura de las paginas web
  • Consiste en una serie de elementos
  • Los elementos le dicen al navegador como mostrar el contenido
  • Los elemnetos tasmbien consisten en piezas de etiquetas con contenido

Basics Elements Explained

  1. La etiqueta "DOCTYOPE" define la declaracion del documento en cuestion como HTML
  2. L etiqueta HTML es el elemento principal de una pagina HTML
  3. La etiqueta "head" contiene informacion adicional acerca de la pagina HTML
  4. La etiqueta "title" especifica el titulo de la pagina web
  5. La etiqueta "body" especifica el cuerpo de la pagina web (todos los elementos visibles)
  6. La etiqueta "h1" define un texto de tamaño titular
  7. La etiqueta "h2" define un texto de tamaño subtitular
  8. La etiqueta "p" define un texto de tamaño parrafo

¿Que es un elemento HTML?

Un elemnteo HTML esta definido por una etiqueta de inicio, seguido de algun contenido y una etiqueta de finalizacion

  • <NombreEtiqueta>: Etiqueta de inicio
  • <Contenido>:entre la etiqueta de inicio y la etiqueta de finalizacion
  • </NombreEtiqueta>: Etiqueta de finalizacion

Etiqueta <MAP >

La etiqueta <map> define el mapa de una imagen.Una imagen mapeada es una imagen que tiene distintas areas detectables por el navegador, logrando identificar de manera que tiene distintas areas detectables por el navigador, logrando identificar de manera independiente cada una de las areas
La etiqueta <map> define las areas de la imagen mediente la etiqueta <area> con los atributos HTML "shape" que define la forma del area señalada en la imagen, "coords" que define las coordenadas de cada uno de los puntos de shape, "href" que puede definir un hipervínculo para cada una de las areas.

Ejemplo:

<img src="workflow.jpg" usemap="#workflowmap">
<map name="workflowmap">
<area shape="rect" coords="34,44,270,350" href="pagina1.com">
<area shape="circle" coords="200,172,333,250" href="pagina2.com">
</map>

Elementos HTML "DIV"

La etiqueta <div> es un elemento que toma el ancho de la pagina por defecto y realiza automaticamente un salto de linea despues de cada elememnto.
<div> se puede interpretar como un contenedor de elemento HTML.
